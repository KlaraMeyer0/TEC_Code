# Thermochemical Equilibrium Dataset

## Data Availability

The thermochemical equilibrium data used in this work was generated using FactSage 8.2 with the FToxide database. The dataset covers quaternary oxide systems in the CaO–MgO–Al₂O₃–SiO₂ system.
